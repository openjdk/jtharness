examples/tutorial/simpleTags/src

This directory contains the custom source code for an example of a simple test suite, built using tag test descriptions. This example is described in and referenced from the JT Harness Architect's Manual.
  • DemoTSInterview.java: the source code for the interview
  • i18n.properties: the default localization strings for the interview
  • moreInfo/: the JavaHelp files containing the "More Info" help for each question in the interview.
